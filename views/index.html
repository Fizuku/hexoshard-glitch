Getting Started
 

GG, you just remixed your own HexoShard!
All you need to do is to get your project domain.
e.g. https://project-name.glitch.me/

 
 

 
This is your database shard.

How can I connect and read data?

Use hexodb, an NPM Package also made by FizxCreations.
It is built for HexoDB Shards.
Using hexo-db will not only allow you to read data, but also read it.

What project domain?

Only your project domain; e.g. https://hexodb.glitch.me/ is needed.
The hexo-db package will let you read, and write data through the database.
Read its documentation by clicking "Docs".
